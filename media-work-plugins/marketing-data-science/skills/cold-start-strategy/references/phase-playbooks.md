# Cold Start Phase Playbooks

## Phase 1: Foundation (Days 1-7)

### Account Setup Checklist

- Profile photo: Consistent branded avatar across all platforms
- Bio: "Daily Claude Code updates in 60 seconds | AI dev tools"
- Link in bio: Landing page or GitHub repo
- Content backlog: Pre-produce 14 videos before first post

### Initial Content Cadence

| Days | TikTok | Instagram | YouTube |
|------|--------|-----------|---------|
| 1-3 | 2x daily | 1x daily | 1x daily |
| 4-7 | 1x daily | 1x daily | 1x daily |

Content type: Best-of CHANGELOG highlights (most impactful features).

### Seeding Strategy

- Cross-post to relevant subreddits (r/ClaudeAI, r/artificial, r/programming)
- Share in Discord communities (Claude, AI dev tools)
- Engage with 20 accounts daily in the AI/dev tools space
- Comment on trending AI content with value-add insights

## Phase 2: Iteration (Days 8-14)

### Content Optimization

- Analyze first week metrics (views, completion rate, shares)
- Double down on top-performing content formats
- A/B test hooks: question vs statement vs demo-first
- Introduce trending audio/formats on TikTok

### Engagement Loops

- Reply to every comment within 2 hours
- Create response videos to questions
- Duet/stitch with relevant AI content creators
- Start "daily Claude Code tip" series

## Phase 3: Growth (Days 15-30)

### Scaling

- Introduce collaboration content with AI dev influencers
- Launch weekly "CHANGELOG recap" longer format
- Create platform-specific series (TikTok challenges, IG carousel tutorials)
- Begin A/B experiment framework (see `ab-experiment-measurement` skill)

### Day 30 Metrics Targets

| Metric | TikTok | Instagram | YouTube |
|--------|--------|-----------|---------|
| Followers | 500+ | 200+ | 100+ |
| Avg views | 1000+ | 300+ | 200+ |
| Completion rate | >40% | >35% | >45% |
| Engagement rate | >5% | >3% | >4% |
