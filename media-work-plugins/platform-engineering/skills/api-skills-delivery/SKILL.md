# API Skills Delivery Reports

Generates professional delivery artifacts using Anthropic managed Skills (xlsx, pptx, pdf).

## API Skills Integration

Platform-engineering uses API Skills for:
- **Excel**: Upload tracking dashboards, rate limit monitoring, platform analytics
- **PDF**: Delivery confirmations, platform setup guides, error reports
- **PowerPoint**: Pipeline status decks, integration architecture diagrams

## Required Setup

```python
# Environment
ANTHROPIC_API_KEY=<your-api-key>

# Python dependencies
pip install anthropic>=0.71.0
```

## Integration Points

### Auto-generate on requirement completion
Platform-engineering hooks trigger report generation when requirements are completed:

```json
{
  "PostToolUse": [{
    "matcher": "Write(requirements/completed/*)",
    "hooks": [{
      "type": "command",
      "command": "python scripts/generate_delivery_report.py"
    }]
  }]
}
```

### Shared output directory
All generated reports are saved to `output/reports/` and referenced in requirement status updates.

## Cost Notes

- API Skills use token-based pricing (same as standard API calls)
- Excel generation: ~2 minutes, ~2000-4000 output tokens
- PowerPoint: ~1-2 minutes, ~2000-4000 output tokens
- PDF: ~40-60 seconds, ~1000-2000 output tokens
- Batch multiple reports when possible to reduce overhead
